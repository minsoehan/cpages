အမိန့်ကြော်ငြာစာအမှတ် - ၆၄/၂၀၁၃

အခန်း (၈)
အနည်းဆုံးအခကြေးငွေ သတ်မှတ်ခြင်း

ကန့်ကွက်နိုင်ရန် အနည်းဆုံး ရက်ပေါင်း (၆၀) ကြိုတင်၍ ထုတ်ပြန်ကြေငြာရခြင်း

၃၄။အမျိုးသားကော်မတီသည် -
(က)အနည်းဆုံးအခကြေးငွေကို နိုင်ငံတစ်ဝန်းလုံးအတွက်သော်လည်းကောင်း၊ ပြည်ထောင်စုနယ်မြေအတွက်သော်လည်းကောင်း၊ တိုင်းဒေသကြီး သို့မဟုတ် ပြည်နယ်အလိုက်သော်လည်းကောင်း၊ ကူးသန်းရောင်းဝယ်ရေးလုပ်ငန်း၊ ကုန်ထုတ်လုပ်ငန်းနှင့် ဝန်ဆောင်မှုလုပ်ငန်း၊ စိုက်ပျိုးရေးနှင့် မွေးမြူရေးလုပ်ငန်း အမျိုးအစားအလိုက်သော်လည်းကောင်း၊ အနည်းဆုံးအခကြေးငွေ သတ်မှတ်ရေးကိစ္စနှင့် စပ်လျဥ်း၍ အထူးစီးပွားရေးဇုန် စီမံခန့်ခွဲရေးကော်မတီက သက်ဆိုင်ရာ တိုင်းဒေသကြီးကော်မတီ သို့မဟုတ် ပြည်နယ်ကော်မတီနှင့်ညှိနှိုင်း၍ တင်ပြလာသည့် အကြံပြုချက်များ၊ ပြည်ထောင်စုနယ်မြေကော်မတီ၊ တိုင်းဒေသကြီးကော်မတီ သို့မဟုတ် ပြည်နယ်ကော်မတီများ၏ အကြံပြုချက်များ၊နှင့် သက်ဆိုင်ရာလုပ်ငန်းကော်မတီ၏ သဘောထားမှတ်ချက်များကို လေ့လာစိစစ်ပြီး သတ်မှတ်သင့်သည့် အဆိုပြုအနည်းဆုံးအခကြေးငွေနှုန်းထားများကို အများပြည်သူ သိရှိစေရန်နှင့် ကန့်ကွက်လိုက ကန့်ကွက်နိုင်ရန် အနည်းဆုံးရက်ပေါင်း (၆၀) ကြိုတင်၍ ထုတ်ပြန်ကြေငြာရမည်။
......
၃၅။ဦးစီးဌာနသည် -
(က)အမျိုးသားကော်မတီ၏ တာဝန်ပေးချက်အရ သတ်မှတ်သော အဆိုပြု အနည်းဆုံးအခကြေးငွေနှုန်းထားများကို အများပြည်သူသိရှိစရန်နှင့် ကန့်ကွက်လိုက ကန့်ကွက်နိုင်ရန် အနည်းဆုံး ရက်ပေါင်း (၆၀) ကြိုတင်၍ ထုတ်ပြန်ပြီး နိုင်ငံတော်ပြန်တမ်းနှင့် သတင်းစာတို့တွင် ထည့်သွင်းကြေညာရမည်။
......
၃၆။အမျိုးသားကော်မတီသည် -
(က)အများပြည်သူသိရှိရန် ကြေငြာပြီးနောက် ရက်ပေါင်း (၆၀) ပြည့်မြောက်သည့်အခါ ကန့်ကွက်ချက်မရှိလျှင် သက်ဆိုင်ရာအနည်းဆုံးအခကြေးငွေနှုန်းထားကို ပြည်ထောင်စု အစိုးရအဖွဲ့၏ အတည်ပြုချက်ဖြင့် သတ်မှတ်ရမည်။
(ခ)နည်းဥပဒေခွဲ (က) အရ သတ်မှတ်သည့် အနည်းဆုံးအခကြေးငွေနှုန်းထားကို အများပြည်သူသိရှိနိုင်ရန် အမိန့်ကြော်ငြာစာကို နိုင်ငံတော်ပြန်တမ်းနှင့် သတင်းစာတို့တွင် ထည့်သွင်းကြေညာရေးတို့အတွက် ဦးစီဌာနကို တာဝန်ပေးရမည်။
၃၇။နည်းဥပဒေ ၃၆ အရ အများပြည်သူသိရှိရန် ကြေညာသည့် အဆိုပြုအနည်းဆုံးအခကြေးငွေနှုန်းထားနှင့် စပ်လျဥ်း၍ ကန့်ကွက်လိုသူ သို့မဟုတ် ကန့်ကွက်လိုသည့်အဖွဲ့အစည်း သို့မဟုတ် ပြင်ဆင်ရန် အဆိုပြုလိုသူ သို့မဟုတ် ပြင်ဆင်ရန်အဆိုပြုလိုသည့် အဖွဲ့အစည်းသည် ပူးတွဲပါပုံစံအတိုင်း ဖြည့်စွက်၍ ရက်သတ္တနှစ်ပတ်အတွင်း ပြည်ထောင်စုနယ်မြေကော်မတီ၊ သက်ဆိုင်ရာ တိုင်းဒေသကြီးကော်မတီ သို့မဟုတ် ပြည်နယ်ကော်မတီသို့ တင်ပြရမည့်အပြင် မိတ္တူကို အမျိုးသားကော်မတီနှင့် ဦးစီဌာနသို့ ပေးပို့ရမည်။
၃၈။ပြည်ထောင်စုနယ်မြေကော်မတီ၊ သက်ဆိုင်ရာတိုင်းဒေသကြီးကော်မတီ သို့မဟုတ် ပြည်နယ်ကော်မတီသည် နည်းဥပဒေ ၃၇ အရ တင်ပြသော အဆိုပြုအနည်းဆုံးအခကြေးငွေနှုန်းထားနှင့် ပတ်သက်၍ ကွင်းဆင်းစစ်ဆေးရန် လို၊ မလို ဆုံးဖြတ်နိုင်သည်။ ကွင်းဆင်းစစ်ဆေးရန် လိုအပ်ပါက -
(က)နည်းဥပဒေ ၃၇ အရ ကန့်ကွက်ခြင်း သို့မဟုတ် ပြင်ဆင်ရန် အဆိုပြုသူအား ကွင်းဆင်းစစ်ဆေးမည့် နေ့ရက်၊ အချိန်နှင့် နေရာတို့ကို သတ်မှတ်ပေးပြီး လိုအပ်သော အချက်အလက်များကောက်ယူရမည်။
(ခ)နည်းဥပဒေခွဲ (က) အရ ရရှိသော အချက်အလက်များနှင့် အထောက်အထားများကို မှတ်တမ်းပြုစုထားရှိရမည်။
(ဂ)အဆိုပါအချက်အလက်များကို ကန့်ကွက်သည့်နေ့ သို့မဟုတ် ပြင်ဆင်ရန်အဆိုပြုသည့်နေ့မှ ရက်ပေါင်း ၃၀ အတွင်း ပြီးစီးအောင် ပြုုလုပ်ရမည်။
(ဃ)ကွင်းဆင်းစစ်ဆေးမှုပြုလုပ်ရာတွင် ကန့်ကွက်သူ သို့မဟုတ် ပြင်ဆင်ရန် အဆိုပြုသူသည် ကိုယ်တိုင်ဖြစ်စေ၊ ကိုယ်စားလှယ်ဖြင့်ဖြစ်စေ၊ လိုက်ပါဆောင်ရွက်ခွင့်ပြုနိုင်သည်။

အခန်း (၉)

အလုပ်ရှင်၏ အခွင့်အရေးနှင့် တာဝန်များ

၄၃။အလုပ်ရှင်သည် -
(က)မိမိလုပ်ငန်း၏ ကုန်ထုတ်စွမ်းအားတိုးတက်စေရေးအတွက် အလုပ်သမားများ၏ အလုပ်အကိုင်ဆိုင်ရာ ကျွမ်းကျင်မှုအဆင့်မြင့်မားစေရန် ဆောင်ရွက်ပေးရမည်ဖြစ်သကဲ့သို့ ကျွမ်းကျင်မှုအလိုက် လုပ်ခလစာတိုးမြှင့်ပေးရမည်။
(ခ)အလုပ်သမားများ၏ စာရင်း၊ ဇယားနှင့် စာတမ်းအမှတ်အသားများ၊ အခကြေးငွေများကို ဥပဒေပုဒ်မ ၁၃(ခ) အရ ပြုစုရာတွင် ၁၉၅၁ ခုနှစ်၊ အလုပ်ရုံများအက်ဥပဒေနှင့် ၁၉၅၁ ခုနှစ်၊ ခွင့်ရက်နှင့် အလုပ်ပိတ်ရက် အက်ဥပဒေတို့နှင့်အညီ ဆောင်ရွက်ရမည်။
(ဂ)အလုပ်သမားများ မကျန်းမာခြင်း၊ လုပ်ငန်းခွင်ထိခိုက်မှုဖြစ်ခြင်းများကြောင့် အလုပ်မလုပ်ကိုင်နိုင်သည့်အခါ -
(၁)ကျန်းမာရေးနှင့် လူမှုရေးစောင့်ရှောက်မှု ရန်ပုံငွေသို့ ထည့်ဝင်ကြေးပေးသွင်းထားသော အားမခံထားသူဖြစ်လျင် ၂၀၁၂ ခုနှစ်၊ လူမှုဖူလုံရေးဥပဒေပါ ကျန်းမာရေးနှင့် လူမှုရေးစောင့်ရှောက်မှု အာမခံစနစ်အရသော်လည်းကောင်း
(၂)၂၀၁၂ ခုနှစ်၊ လူမှုဖူလုံရေးဥပဒေနှင့် အကျုံးမဝင်သော အလုပ်သမားဖြစ်လျှင် ၁၉၅၁ ခုနှစ်၊ ခွင့်ရက်နှင့် အလုပ်ပိတ်ရက်ဥပဒေအရသော်လည်းကောင်း ဆောင်ရွက်ပေးရမည်။
(ဃ)အလုပ်သမားများ၏ မိသားစု သို့မဟုတ် မိဘနာရေးကိစ္စဖြစ်သည့်အခါ ၎င်းအလုပ်သမား ရထိုက်သောအခကြေးငွေမှ ဖြတ်တောက်ခြင်းမပြုဘဲ ၁၉၅၁ ခုနှစ်၊ ခွင့်ရက်နှင့် အလုပ်ပိတ်ရက် အက်ဥပဒေအရ ဆောင်ရွက်ပေးရမည်။
(င)ဤဥပဒေအရ အမျိုးသားကော်မတီက အနည်းဆုံးအခကြေးငွေသတ်မှတ်ခြင်းမပြုမီ အလုပ်သမားရရှိနေသော အခကြေးငွေသည် သတ်မှတ်ချက်အောက်လျော့နည်းနေပါက သတ်မှတ်ချက်ပြည့်မီသည်အထိ ပေးရမည်။
(စ)အချိန်ပိုင်း၊ နာရီပိုင်းအလုပ်တွင် အလုပ်လုပ်သည့် အလုပ်သမားအား အလုပ်လုပ်သည့် အချိန်အတွက် သတ်မှတ်ထားသည့် အနည်းဆုံးအခကြေးငွေကို ပေးရမည်။
(ဆ)လစားအလုပ်သမားအတွက် ရက်သတ္တပတ်လျှင် တစ်ရက်နားခွင့်ပေးရမည်။ ယင်းနားရက်တွင် အလုပ်လုပ်ရပါက တည်ဆဲဥပဒေနှင့်အညီ အချိန်ပိုလုပ်ခပေးရမည်။
(ဇ)သတ်မှတ်ထားသော အလုပ်ချိန်ထက်လျော့၍ လုပ်ရပါက ထိုသို့အလုပ်လုပ်ခြင်းမှာ အလုပ်သမား၏ ဆန္ဒအရမဟုတ်လျှင်သော်လည်းကောင်း၊ အလုပ်ရှင်က အလုပ်မပေးနိုင်၍ အလုပ်နားရလျှင်သော်လည်းကောင်း အချိန်ပြည့်အလုပ်လုပ်ဘိသကဲ့သို့ အခကြေးငွေပေးရမည်။
(ဈ)သတ်မှတ်ထားသော အနည်းဆုံး အခကြေးငွေကို အမျိုးသား၊ အမျိုးသမီး မခွဲခြားဘဲ ပေးရမည်။

(က)ပုဒ်မ ၈ နှင့် ပုဒ်မ ၉ တို့အရ အကြံပြုချက်အပေါ် စဥ်းစားသုံးသပ်ပြီးနောက် အနည်းဆုံးအခကြေးငွေသတ်မှတ်မည့် အဆိုပြုအမိန့်ကြော်ငြာစာကို အများပြည်သူသိရှိစေရန်နှင့် ကန့်ကွက်လိုက ကန့်ကွက်နိုင်ရန် အနည်းဆုံး ရက်ပေါင်း ၆၀ ကြိုတင်၍ နိုင်ငံတော်ပြန်တမ်းနှင့် သတင်းစာများတွင် ထည့်သွင်းထုတ်ပြန်ကြေညာရေးနှင့် သက်ဆိုင်ရာ အဖွဲ့အစည်းများ၊ အလုပ်ရှင်များနှင့် အလုပ်သမားများအား အသိပေးအကြောင်းကြားရေးတို့အတွက် ဦးစီးဌာနကို တာဝန်ပေးရမည်။
(ခ)အဆိုပြု အနည်းဆုံးအခကြေးငွေနှုန်းထားအပေါ် ကန့်ကွက်ချက်မရှိလျှင် ပြည်ထောင်စုအစိုးရအဖွဲ့၏ အတည်ပြုချက်ရယူ၍ အနည်းဆုံးအခကြေးငွေနှုန်းထားကို သတ်မှတ်ရမည်။
(ဂ)အကယ်၍ အဆိုပြု အနည်းဆုံးအခကြေးငွေနှုန်းထားအပေါ် ကန့်ကွက်ချက်ရှိသည့်ကိစ္စတွင် ပြည်ထောင်စုနယ်မြေကော်မတီ၊ တိုင်းဒေသကြီးကော်မတီ သို့မဟုတ် ပြည်နယ်ကော်မတီအား ကန့်ကွက်ချက်နှင့် စပ်လျဥ်း၍ ညှိနှိုင်းစေပြီး အဆိုပြုနှုန်းထားကို ပြန်လည်တင်ပြစေရမည်။

၁၁။ပုဒ်မ ၉ နှင့် ၁၀ တို့အရ အမျိုးသားကော်မတီက သတ်မှတ်သော အနည်းဆုံးအခကြေးငွေနှုန်းထားကို ကျေနပ်မှုမရှိသူသည် ပြည်ထောင်စုသမ္မတ မြန်မာနိုင်ငံတော် ဖွဲ့စည်းပုံအခြေခံဥပဒေနှင့် ပြည်ထောင်စုတရားစီရင်ရေး ဥပဒေတို့နှင့်အညီ ပြည်ထောင်စုတရားလွှတ်တော်ချုပ်သို့ စာချွန်တော်အမိန့်တစ်ရပ်ရပ် ထုတ်၍ ပြင်ဆင်ပေးရန် လျှောက်ထားနိုင်သည်။

အခန်း (၇)

အလုပ်ရှင်တာဝန်များ

၁၂။အလုပ်ရှင်သည် -
(က)ဤဥပဒေအရ သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေအောက် လျော့နည်း၍ အလုပ်သမားအား အခကြေးငွေပေးခြင်း မပြုရ။
(ခ)ဤဥပဒေအရ သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေထက် ပို၍ပေးနိုင်သည်။
(ဂ)ဤဥပဒေအရ ထုတ်ပြန်သော အမိန့်ကြော်ငြာစာတွင် အနည်းဆုံး အခကြေးငွေမှ နှုတ်ယူခွင့်ရှိသည်ဟု သတ်မှတ်ထားသော အခကြေးငွေမှအပ အခြားအခကြေးငွေကို နှုတ်ယူခွင့်မရှိစေရ။
(ဃ)ကူးသန်းရောင်းဝယ်ရေးလုပ်ငန်း၊ ကုန်ထုတ်လုပ်ငန်းနှင့် ဝန်ဆောင်မှုလုပ်ငန်းများတွင် လုပ်ကိုင်နေသော အလုပ်သမားအား အနည်းဆုံးအခကြေးငွေကို ငွေသားဖြင့် ပေးရမည်။ ထို့ပြင် သီးခြားအကျိုးခံစားခွင့်များ၊ အကျိုးအမြတ်များ၊ အခွင့်အလမ်းများကို ပေးရန်ရှိပါက သတ်မှတ်ချက်များနှင့်အညီ ငွေသားဖြင့်ဖြစ်စေ၊ အလုပ်သမား၏ ဆန္ဒအရ ငွေသားတစ်ချို့တစ်ဝက်နှင့် ဒေသပေါက်ဈေးနှုန်းဖြင့်သတ်မှတ်သော ပစ္စည်းတစ်ချို့တစ်ဝက်တွဲဖက်၍ ပေးနိုင်သည်။
(င)စိုက်ပျိုးရေးနှင့် မွေးမြူရေးလုပ်ငန်းတွင် လုပ်ကိုင်သော အလုပ်သမားများအား အနည်းဆုံးအခကြေးငွေပေးရာတွင် ဒေသဓလေ့ထုံးတမ်းအရဖြစ်စေ၊ အလုပ်သမားအများစု၏ ဆန္ဒအရဖြစ်စေ၊ စုပေါင်းသဘောတူညီချက်အရဖြစ်စေ၊ ငွေသားတစ်ချို့တစ်ဝက်နှင့် ဒေသပေါက်ဈေးနှုန်းဖြင့်သတ်မှတ်သော ပစ္စည်းတစ်ချို့တစ်ဝက်တွဲဖက်၍ ပေးနိုင်သည်။ ထိုသို့ပေးခြင်းသည် အလုပ်သမားနှင့် ယင်း၏မိသားစု ကိုယ်ရေးကိုယ်တာ အသုံးပြုမှုနှင့် အကျိုးအမြတ်တစ်စုံတစ်ရာအတွက် ဖြစ်ရမည့်အပြင် ပေးသည့်တန်ဖိုးမှာလည်း ကျိုးကြောင်းဆီလျော်ပြီး မျှတရမည်။
၁၃။အလုပ်ရှင်သည် -
(က)ဤဥပဒေအရ သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေ နှုန်းထားများအနက် လုပ်ငန်းနှင့် သက်ဆိုင်သော အနည်းဆုံးအခကြေးငွေ နှုန်းထားများကို အလုပ်သမားများအား အသိပေးရမည့်အပြင် လုပ်ငန်းခွင်တွင် သက်ဆိုင်ရာ အလုပ်သမားများ မြင်နိုင်စေရန် ကြော်ငြာထားရမည်။
(ခ)အလုပ်သမားများ၏ စာရင်း၊ ဇယားနှင့် စာတမ်းအမှတ်အသားများ၊ အခကြေးငွေများကို မှန်ကန်စွာမှတ်တမ်းပြုစုထားရှိရမည်။
(ဂ)ပုဒ်မခွဲ (ခ) အရ ပြုစုထားရှိသော စာရင်း၊ ဇယားနှင့် စာတမ်းအမှတ်အသားများကို သတ်မှတ်ချက်များနှင့်အညီ သက်ဆိုင်ရာ ဦးစီးဌာနကို အစီရင်ခံရမည်။
(ဃ)စစ်ဆေးရေးအရာရှိက စစ်ဆေးရန် ဆင့်ဆိုသောအခါ သွားရောက်စစ်ဆေးခံရမည်။ ထို့အပြင် အဆိုပါ စာရင်းနှင့် မှတ်တမ်းများကို တောင်းခံသည့်အခါ တင်ပြရမည်။
(င)ကူးသန်းရောင်းဝယ်ရေးလုပ်ငန်း၊ ကုန်ထုတ်လုပ်ငန်းနှင့် ဝန်ဆောင်မှုလုပ်ငန်းများ၊ စိုက်ပျိုးရေးနှင့် မွေးမြူရေးလုပ်ငန်းခွင်များသို့ စစ်ဆေးရေးအရာရှိက ဝင်ရောက်စစ်ဆေးခြင်းကို ခွင့်ပြုရမည်ဖြစ်ပြီး လိုအပ်သော အကူအညီများ ပေးရမည်။
(စ)အလုပ်သမားများ ဖျားနာ၍ အလုပ်မလုပ်နိုင်သည့်အခါ သတ်မှတ်ချက်များနှင့်အညီ ဆေးကုသရန် အလုပ်နားခွင့်ပေးရမည်။
(ဆ)အလုပ်သမားများ၏ မိသားစုဝင် သို့မဟုတ် မိဘနာရေးကိစ္စ ဖြစ်သည့်အခါ အနည်းဆုံးအခကြေးငွေမှ ဖြတ်တောက်ခြင်းမပြုဘဲ သတ်မှတ်ချက်များနှင့်အညီ အလုပ်နားခွင့်ပေးရမည်။

အခန်း (၈)

အနည်းဆုံးအခကြေးငွေနှင့် စပ်လျဥ်း၍ အလုပ်သမားများ၏ အခွင့်အရေးများ

၁၄။ဤဥပဒေနှင့် သက်ဆိုင်သည့် လုပ်ငန်းတစ်ခုခုတွင် အလုပ်လုပ်နေသော အလုပ်သမားသည် -
(က)ဤဥပဒေအရ သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေကိုသော်လည်းကောင်း၊ ထိုအခကြေးငွေထက် အလုပ်ရှင်က ပို၍ပေးပါက ပိုပေးသည့် အခကြေးငွေကိုသော်လည်းကောင်း ရထိုက်ခွင့်ရှိသည်။
(ခ)ဤဥပဒေ အာဏာမတည်မီက ရရှိနေသော အခကြေးငွေသည် ဤဥပဒေအရ သတ်မှတ်သော အနည်းဆုံးအခကြေးငွေထက် ပိုမိုရရှိပါက အခကြေးငွေကို ဆက်လက်ခံစားခွင့်ရှိသည်။
(ဂ)အလုပ်ခန့်ထားမှုဆိုင်ရာ သဘောတူညီချက်ပါ အနည်းဆုံးအခကြေးငွေမှာ ဤဥပဒေအရ သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေထက် လျှော့နည်းနေပါက ဤဥပဒေအရ သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေကို ရထိုက်ခွင့်ရှိသည်။
(ဃ)အလုပ်နှစ်ခု သို့မဟုတ် နှစ်ခုထက်ပိုသော အလုပ်ကို လုပ်ကိုင်ရသည့်အခါ လုပ်ကိုင်သော အလုပ်အသီးသီးအတွက် သတ်မှတ်ထားသော အနည်းဆုံးအခကြေးငွေထက် မနည်းစေဘဲ ခံစားခွင့်ရှိသည်။
(င)အချိန်ပိုင်း၊ နာရီပိုင်း အလုပ်တွင် အလုပ်လုပ်သည့်အချိန်အတွက် သတ်မှတ်ထားသည့် အနည်းဆုံးအခကြေးငွေကို ခံစားခွင့်ရှိသည်။
(စ)လစားအလုပ်အတွက် ရက်သတ္တတစ်ပတ်လျှင် တစ်ရက် နားခွင့်ရှိပြီး ထိုနားရက်အတွက် အခကြေးငွေ ရထိုက်ခွင့်ရှိသည်။ အကယ်၍ ထိုနားရက်တွင် အလုပ်လုပ်ရပါက တည်ဆဲဥပဒေနှင့်အညီ အချိန်ပိုလုပ်ခ ရထိုက်ခွင့်ရှိသည်။
(ဆ)နေ့စားအလုပ်တွင် သတ်မှတ်ထားသည့် တစ်နေ့အလုပ်ချိန်နာရီထက်လျော့၍ အလုပ်လုပ်ခြင်းမှာ အလုပ်သမား၏ ဆန္ဒအရမဟုတ်လျှင်သော်လည်းကောင်း၊ အလုပ်ရှင်က အလုပ်မပေးနိုင်၍ အလုပ်နားရလျှင်သော်လည်းကောင်း အချိန်ပြည့် အလုပ်လုပ်ဘိသကဲ့သို့ အခကြေးငွေ ရထိုက်ခွင့်ရှိသည်။
(ဇ)သတ်မှတ်ထားသည့် အနည်းဆုံးအခကြေးငွေကို အမျိုးသား၊ အမျိုးသမီး မခွဲခြားဘဲ ခံစားခွင့်ရှိစေရမည်။

(က)အခကြေးငွေဆိုသည်မှာ အလုပ်ရှင်၏ နာရီပိုင်းအလုပ်၊ နေ့စဥ်အလုပ်၊ အပတ်စဥ်အလုပ်၊ လစဥ်အလုပ် သို့မဟုတ် အခြားအချိန်ပိုင်းအလုပ်တစ်ခုခုကို အလုပ်သမားက လုပ်ကိုင်ဆောင်ရွက်ပေးရသဖြင့် ရသင့်သော အခကြေးငွေနှင့် လုပ်ခလစာတို့ကို ဆိုသည်။ ယင်းစကားရပ်တွင် အချိန်ပိုလုပ်ခသည်လည်းကောင်း၊ အလုပ်သမား၏ စွမ်းဆောင်ရည်ကောင်းမွန်သဖြင့် သို့မဟုတ် အကျင့်စာရိတ္တကောင်းမွန်သဖြင့် အလုပ်ရှင်က အပိုပေးသည့် ဆုကြေးငွေများသည်လည်းကောင်း၊ ဝင်ငွေအဖြစ် သတ်မှတ်နိုင်သော အခြားငွေနှင့် အကျိုးခံစားခွင့်များသည်လည်းကောင်း ပါဝင်သည်။ သို့ရာတွင် အောက်ပါအတို့ မပါဝင်။
(၁)ခရီးစရိတ် သို့မဟုတ် ခရီးသွားလာမှုအတွက် အထူးထောက်ပံ့စရိတ်များ
(၂)လုပ်ငန်းသဘာဝအရ နှစ်ဖက်ကြိုတင်ညှိနှိုင်းထား၍ ခေတ္တစိုက်ထုတ်ကျခံသုံးစွဲငွေများ
(၃)လူမှုဖူလုံရေးအကျိုးခံစားခွင့်ငွေကြေးများ
(၄)တည်ဆဲဥပဒေတစ်ရပ်ရပ်အရ အလုပ်သမားအတွက် ပေးသွင်းသည့် ထည့်ဝင်ကြေးများ
(၅)နေထိုင်ခနှင့် စားသောက်စရိတ်များ၊ လျှပ်စစ်ဓါတ်အားခ၊ ရေဖိုးနှင့် အခွန်အခများ
(၆)ဆေးကုသစရိတ်နှင့် အပန်းဖြေခများ
(၇)အလုပ်ထုတ်နစ်နာကြေးနှင့် ကရုဏာကြေးများ
(၈)ပင်စင်လစာနှင့် အငြိမ်းစားလုပ်သက်ဆုငွေ
(၉)ဝန်ကြီးဌာနက ဤဥပဒေပါအခကြေးငွေနှင့် မသက်ဆိုင်ဟု ပြည်ထောင်စုအစိုးရအဖွဲ့၏ အတည်ပြုချက်ဖြင့် အမိန့်ကြော်ငြာစာထုတ်ပြန်၍ သတ်မှတ်သော အခြားငွေကြေးများ။

အခန်း (၂)
အခကြေးငွေပေးရမည့် နည်းလမ်းနှင့် အချိန်ကာလ

အခကြေးငွေပေးရမည့်နည်းလမ်း

၃။အလုပ်ရှင်သည် -
(က)မိမိ၏ လုပ်ငန်းတွင် လုပ်ကိုင်နေသော အလုပ်သမား၏ အခကြေးငွေကို ပြည်တွင်းသုံးငွေဖြင့်ဖြစ်စေ၊ မြန်မာနိုင်ငံတော်ဗဟိုဘဏ်က အသိအမှတ်ပြုသည့်နိုင်ငံခြားသုံးငွေဖြင့်ဖြစ်စေ ပေးချေရမည်။ ထိုသို့ ပေးချေရာတွင် အလုပ်ရှင်၊ အလုပ်သမားနှစ်ဘက် သဘောတူညီမှုဖြင့် ငွေသားဖြင့်ဖြစ်စေ၊ ချက်လက်မှတ်ဖြင့်ဖြစ်စေ၊ အလုပ်သမား၏ ဘဏ်စာရင်းသို့ဖြစ်စေ ပေးချေနိုင်သည်။
(ခ)အခကြေးငွေပေးချေရာတွင် -
(၁)ကူးသန်းရောင်းဝယ်ရေးလုပ်ငန်း၊ ကုန်ထုတ်လုပ်ငန်းနှင့် ဝန်ဆောင်မှုလုပ်ငန်းများတွင် လုပ်ကိုင်နေသော အလုပ်သမားများအား သီးခြားအကျိုးခံစားခွင့်များ၊ အကျိုးအမြတ်များ၊ အခွင့်အလမ်းများကို ပေးချေရန်ရှိပါက သတ်မှတ်ချက်များနှင့်အညီ ငွေးသားဖြင့်ဖြစ်စေ၊ အလုပ်သမား၏ ဆန္ဒအရ ငွေသားတစ်ချို့တစ်ဝက်နှင့် တွဲဖက်၍ဖြစ်စေ ပေးချေနိုင်သည်။
(၂)စိုက်ပျိုးရေးနှင့် မွေးမြူရေးလုပ်ငန်းတွင် လုပ်ကိုင်သော အလုပ်သမားများအား ဒေသဓလေ့ထုံးတမ်းအရဖြစ်စေ၊ အလုပ်သမားအများစု၏ ဆန္ဒအရဖြစ်စေ၊ စုပေါင်းသဘောတူညီချက်အရဖြစ်စေ၊ ငွေးသားတစ်ချို့တစ်ဝက်နှင့် ဒေသပေါက်ဈေးနှုန်းဖြင့် သတ်မှတ်သော ပစ္စည်းတစ်ချို့ တစ်ဝက်တွဲဖက်၍ ပေးချေနိုင်သည်။ ထိုသို့ပေးချေခြင်းသည် အလုပ်သမားနှင့် ယင်းမိသားစုကိုယ်ရေးကိုယ်တာ အသုံးပြုမှုနှင့် အကျိုးအမြတ်တစ်စုံတစ်ရာအတွက် ဖြစ်ရမည့်အပြင် ပေးချေသည့် တန်ဖိုးမှာလည်း ကျိုးကြောင်းဆီလျော်ပြီး မျှတရမည်။
(ဂ)အလုပ်သမားတစ်ဦးဦးက ပြည်သူ့စစ်မှုထမ်းဥပဒေအရ စစ်မှုထမ်းရသည့်အခါ ထိုအလုပ်သမားအား အထူးခွင့်အဖြစ် သတ်မှတ်ပြီး ရက်ပေါင်း (၆၀) အတွက် အခကြေးငွေထုတ်ပေးရမည်။

အခကြေးငွေပေးရမည့်အချိန်ကာလ

၄။အလုပ်ရှင်သည် -
(က)နာရီပိုင်းအလုပ်၊ နေ့စဥ်အလုပ်၊ အပတ်စဥ်အလုပ် သို့မဟုတ် အခြားအချိန်ပိုင်းအလုပ် တစ်ခုခုအတွက်သော်လည်းကောင်း၊ ယာယီအလုပ် သို့မဟုတ် ပုတ်ပြတ်အလုပ်တစ်ခုခုအတွက်သော်လည်းကောင်း အလုပ်ပြီးဆုံးသည့်အခါဖြစ်စေ၊ အလုပ်သမားနှင့် သဘောတူညီသည့် အချိန်ကာလအတွင်းဖြစ်စေ အခကြေးငွေပေးချေရမည်။
(ခ)ပုဒ်မခွဲ (က) အရ အခကြေးငွေပေးချေရန် အလုပ်သမားနှင့် သဘောတူညီသည့် အချိန်ကာလအပိုင်းအခြားသည် တစ်လထက် မကျော်စေရ။
(ဂ)အမြဲတမ်းအလုပ်အတွက် အခကြေးငွေကို လစဥ်ပေးချေရမည်။ ထိုသို့ပေးချေရာတွင် -
(၁)အလုပ်သမားဦးရေ ၁၀၀ ထက်မပိုပါက အခကြေးငွေပေးရန် ကာလအပိုင်းအခြား ကုန်ဆုံးသည့်နေ့တွင် ပေးချေရမည်။
(၂)အလုပ်သမားဦးရေ ၁၀၀ ထက်ပိုပါက အခကြေးငွေပေးရန် ကာလအပိုင်းအခြား ကုန်ဆုံးပြီးနောက် နောက်အကျဆုံး ငါးရက်အတွင်း ပေးချေရမည်။
(ဃ)အလုပ်သမားအား အလုပ်ရပ်စဲသည့်အခါ ပေးထိုက်သည့် အခကြေးငွေကို အလုပ်မှ ရပ်စဲခြင်းခံရသည့်နေ့မှ အလုပ်ဖွင့်ရက် နှစ်ရက်အတွင်း ပေးချေရမည်။
(င)အလုပ်သမားက အလုပ်ရှင်ထံ ကြိုတင်အကြောင်းကြားစာ ပေးပို့၍ မိမိဆန္ဒအရ အလုပ်မှနှုတ်ထွက်ပါက အခကြေးငွေပေးချေရန် ကာလအပိုင်းအခြား ကုန်ဆုံးသည့်နေ့တွင် ပေးချေရမည်။
(စ)အလုပ်သမားသေဆုံးသည့်အခါ ထိုအလုပ်သမားအား ပေးထိုက်သည့် အခကြေးငွေကို သေဆုံးပြီးနောက် အလုပ်ဖွင့်ရက်နှစ်ရက်အတွင်း သေဆုံးသူ၏ တရားဝင်အမွေဆက်ခံသူထံသို့ ပေးချေရမည်။
(ဆ)အခကြေးငွေအားလုံးကို အလုပ်ဖွင့်သည့်နေ့ရက်တွင် ပေးချေရမည်။
၅။အလုပ်ရှင်သည် သဘာဝဘေးအန္တရာယ်အပါအဝင် မမျှော်လင့်သော ထူးခြားသည့် အခြေအနေတစ်ရပ်ရပ်ပေါ်ပေါက်သဖြင့် ပုဒ်မ ၄၊ ပုဒ်မခွဲ (ဂ) ပါ ပြဌာန်းချက်နှင့်အညီ ပေးချေရန် အခက်အခဲရှိပါက အလုပ်သမား၏ သဘောတူညီချက်ဖြင့် အခကြေးငွေကို မည်သည့်နေ့ရက်သို့ ပြောင်းလဲပေးချေမည်ဖြစ်ကြောင်း ကျိုးကြောင်းခိုင်လုံစွာဖြင့် ဦးစီးဌာနသို့ တင်ပြရမည်။
၆။ဦးစီးဌာနသည် ပုဒ်မ ၅ အရ တင်ပြချက်ကို စိစစ်ပြီး ခွင့်ပြုသင့်ပါက ဝန်ကြီးဌာန၏ သဘောတူညီချက်ဖြင့် စည်းကမ်းချက်များ သတ်မှတ်ပြီး သင့်လျော်သော အချိန်ကာလအတွင်း ရွှေ့ဆိုင်းပေးချေရန် အလုပ်ရှင်အား ခွင့်ပြုနိုင်သည်။

အခန်း (၃)
အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ခြင်း

အလုပ်မဆင်းသည့်ကာလ၊ အလုပ်ရှင်ဘက်မှပံ့ပိုးမှု၊ ကြိုတင်ထုတ်ပေးငွေ၊ စိုက်ထုတ်ပေးငွေ၊ စုငွေ၊ ဥပဒေအရ ထည့်ဝင်ကြေးနှင့် တရားရုံး သို့မဟုတ် ခုံသမာဓိကောင်စီ သို့မဟုတ် ခုံသမာဓိအဖွဲ့၏ ဆုံးဖြတ်ချက်အရ တို့အတွက် ဖြတ်တောက်ခြင်း

၇။အလုပ်ရှင်သည် -
(က)သက်ဆိုင်ရာဥပဒေအရ အခကြေးငွေ ခံစားခွင့်ရှိသည့် ခွင့်ရက်များနှင့် အများပြည်သူအလုပ်ပိတ်ရက်များမှအပ အလုပ်မဆင်းသည့် ကာလအတွက် အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်နိုင်သည်။
(ခ)အခကြေးငွေတွင် ပါဝင်ခြင်းမရှိသည့် အလုပ်ရှင်မှ စီစဥ်ပေးသည့် နေထိုင်ခနှင့် ပို့ကြိုယာဥ်စရိတ်၊ စားသောက်စရိတ်များ၊ လျှပ်စစ်ဓါတ်အားခ၊ ရေဖိုးနှင့် အလုပ်သမားက ပေးဆောင်ရမည့်ဝင်ငွေခွန်နှင့် မှားယွင်း၍ ပိုပေးမိသော အခကြေးငွေများကို နှုတ်ယူဖြတ်တောင်နိုင်သည်။
(ဂ)အလုပ်သမား၏ တောင်းဆိုချက်အရ ကြိုတင်ထုတ်ပေးငွေ၊ စိုက်ထုတ်ပေးငွေ၊ အလုပ်သမားအတွက် စုငွေ သို့မဟုတ် ဥပဒေအရ ထည့်ဝင်ကြေးငွေတစ်ရပ်ရပ်ကို အခကြေးငွေမှ နှုတ်ယူဖြတ်တောင်နိုင်သည်။
(ဃ)တရားရုံး သို့မဟုတ် ခုံသမာဓိကောင်စီ သို့မဟုတ် ခုံသမာဓိအဖွဲ့၏ ဆုံးဖြတ်ချက်အရ အလုပ်သမား၏ အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်နိုင်သည်။

ပုဒ်မ ၇ နှင့် ၁၁ တို့မှအပ အခကြးငွေမှ ဖြတ်တောက်၍မရခြင်း

၈။အလုပ်ရှင်သည် ပုဒ်မ ၇ နှင့် ပုဒ်မ ၁၁ တို့ပါ ပြဌာန်းချက်များနှင့်အညီ အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ခြင်းမှအပ မည်သည့်ကိစ္စအတွက်မျှ နှုတ်ယူဖြတ်တောက်ခြင်းမပြုရ။

ပုဒ်မ ၇ အရ အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ရာတွင် အလုပ်သမားရရှိသည့် အခကြေးငွေ၏ ၅၀ ရာခိုင်နှုန်းထက် မပိုစေရခြင်း

၉။အလုပ်ရှင်သည် ပုဒ်မ ၇ အရ အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ရာတွင် အလုပ်သမားက တာဝန်ထမ်းဆောင်ရန် ပျက်ကွက်မှုအတွက် နှုတ်ယူဖြတ်တောက်ငွေများမှအပ အခြားနှုတ်ယူဖြတ်တောက်ငွေ စုစုပေါင်းသည် အလုပ်သမားရရှိသည့် အခကြေးငွေ၏ ၅၀ ရာခိုင်နှုန်းထက် မပိုစေရ။

အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ခြင်းအတွက် လိုက်နာဆောင်ရွက်ရမည့် အချက်များ

၁၀။အလုပ်ရှင်သည် -
(က)ပုဒ်မ ၁၁ အရ ပေးလျော်ရန် ဒဏ်ငွေအဖြစ် သတ်မှတ်ပြိး အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ခြင်းမပြုမီ မည်သည့်ကိစ္စအတွက် မည်သို့မည်မျှ ဖြတ်တောက်မည်ကို ဦးစီးဌာန၏ ကြိုတင်ခွင့်ပြုချက် ရယူရမည်။
(ခ)ပုဒ်မခွဲ (က) ပါခွင့်ပြုချက်ကို သက်ဆိုင်ရာအလုပ်ရုံ၊ လုပ်ငန်းဌာန၌ အများမြင်သာသော နေရာများတွင် ကပ်ထားရမည်။
(ဂ)နှုတ်ယူဖြတ်တောက်သည့် ပေးလျော်ရန်ဒဏ်ငွေမှာ အလုပ်သမား၏ လုပ်ဆောင်မှု သို့မဟုတ် လုပ်ဆောင်ရန်ပျက်ကွက်မှုကြောင့် ပျက်စီးမှု သို့မဟုတ် ဆုံးရှုံးမှု တန်ဖိုးထက် မပိုစေရ။
(ဃ)ပုဒ်မ ၁၁ အရ အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ရာတွင် -
(၁)အလုပ်သမားအား ထုချေဖြေရှင်းခွင့် တစ်စုံတစ်ရာမပေးဘဲ ထိုအခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်ခြင်း မပြုရ။
(၂)အလုပ်သမား၏ လစဥ်အခကြေးငွေမှ တစ်လလျှင် ၅ ရာခိုင်နှုန်းထက်ပို၍ နှုတ်ယူဖြတ်တောက်ခြင်းမပြုရ။
(င)အသက် ၁၆ နှစ်အောက် အလုပ်သမားထံမှ ဒဏ်ငွေအဖြစ် လုံးဝနှုတ်ယူဖြတ်တောက်ခြင်း မပြုရ။
(စ)ချမှတ်ထားသော ဒဏ်ငွေအား ပေးချေရမည့် ကာလသတ်မှတ်ချက်ကို အလုပ်ရှင်နှင့် အလုပ်သမား သဘောတူညီချက်နှင့်အညီ ဆောင်ရွက်နိုင်သည်။
(ဆ)ပစ္စည်းဆုံးရှုံးမှုအတွက် ဖြတ်တောက်ငွေကို ဥပဒေအရ ဖွဲ့စည်းထားသည့် မြို့နယ် ညှိနှိုင်းဖျန်ဖြေရေးအဖွဲ့ဖြင့် ညှိနှိုင်းသဘောတူညီချက်အရ သတ်မှတ်အချိန်ကာလအတွင်း နှုတ်ယူဖြတ်တောက်ရမည်။
(ဇ)အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်သည့်ငွေများကို မှတ်တမ်းတွင် ရေးသွင်း၍ စနစ်တကျ ထိန်းသိမ်းထားရမည်။
(ဈ)အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်သည့်ငွေနှင့် စပ်လျဥ်း၍ လစဥ်အစီရင်ခံစာကို ဦးစီးဌာနသို့ တင်ပြရမည်။
(ည)ပုဒ်မ ၁၁၊ ပုဒ်မခွဲ (ခ) အရ နှုတ်ယူဖြတ်တောက်ခြင်းသည့် ဒဏ်ငွေများကို စက်ရုံရှိ တရားဝင်မှတ်ပုံတင်ထားသော အလုပ်သမားအဖွဲ့အစည်းနှင့် ညှိနှိုင်း၍ အလုပ်သမားများ အကျိုးဖြစ်ထွန်းစေမည့် ကိစ္စရပ်များ၌ အသုံးပြုရမည်။

လျော်ကြေးငွေ သို့မဟုတ် ဒဏ်ငွေ အဖြစ် နှုတ်ယူဖြတ်တောက်ခြင်း

၁၁။အလုပ်ရှင်သည် အလုပ်သမား၏ အောက်ဖော်ပြပါ ပြုလုပ်မှု သို့မဟုတ် ပျက်ကွက်မှုအတွက် ပေးလျော်ရန်ဒဏ်ငွေအဖြစ် သတ်မှတ်ပြီး အခကြေးငွေမှ နှုတ်ယူဖြတ်တောက်နိုင်သည်။
(က)အလုပ်သမား၏ တမင်လစ်လျူရှုမှု၊ ပေါ့ဆမှုကြောင့်ဖြစ်စေ၊ မရိုးဖြောင့်သော သဘောဖြင့် ပြုလုပ်မှု သို့မဟုတ် ပျက်ကွက်မှုကြောင့်ဖြစ်စေ၊ အလုပ်ရှင်က အလုပ်သမားအား ထိန်းသိမ်းစောင့်ရှောက်ရန် အတိအလင်းအပ်နှံထားသည့် တာဝန်ရှိသော ပစ္စည်းနှင့် ငွေကြေးတစ်စုံတစ်ရာ ပျက်စီးဆုံးရှုံးမှုသည် ထိုအလုပ်သမား၏ ပေါ့ဆမှားယွင်းမှုကြောင့် တိုက်ရိုက်ပေါ်ပေါက်လာသည့် ပျက်စီးဆုံးရှုံးမှု
(ခ)အလုပ်ခန့်ထားမှုဆိုင်ရာ သဘောတူညီချက်စာချုပ်ပါ ဒဏ်ငွေသတ်မှတ်ထားသည့် အလုပ်ခွင် စည်းကမ်းချက်တစ်ရပ်ရပ်ကို အလုပ်သမားက ဖောက်ဖျက်မှု။

အလုပ်သမားဘက်မှ တောင်းဆိုနိုင်ခြင်း

၁၂။အလုပ်သမားသည် -
(က)အောက်ပါအခြေအနေများပေါ်ပေါက်လျှင် ကိုယ်တိုင်ဖြစ်စေ၊ တရားဝင် မှတ်ပုံတင်ထားသော အလုပ်သမားအဖွဲ့အစည်းကဖြစ်စေ၊ စက်ရုံတွင်း လုပ်ငန်းညှိနှုိင်းရေး ကော်မတီကဖြစ်စေ ဖြေရှင်းပေးရန် အလုပ်ရှင်သို့ တောင်းဆိုနိုင်သည်။
(၁)ရရန်ရှိသည့် အခကြေးငွေထဲမှာ ခိုင်လုံသော အကြောင်းပြချက်မရှိဘဲ ဖြတ်တောက်ခံရခြင်း။
(၂)အခကြေးငွေများကို ပေးချေရမည့် ကာလထက်ကျော်လွန်၍ ပေးချေရန်ပျက်ကွက်နေခြင်း။
(ခ)ပုဒ်မခွဲ (က) အရ တောင်းဆိုသော်လည်း အလုပ်ရှင်က ဖြေရှင်းဆောင်ရွက်ပေးခြင်းမရှိပါက နှုတ်ယူဖြတ်တောက်သည့်နေ့ သို့မဟုတ် ပေးချေရန်ပျက်ကွက်သည့်နေ့မှ ခြောက်လအတွင်း စစ်ဆေးရေးအရာရှိထံသို့ ဖြေရှင်းဆောင်ရွက်ပေးရန် တင်ပြနိုင်သည်။

(ခ)အလုပ်ရှင် သို့မဟုတ် အလုပ်သမားသည် ပုဒ်မခွဲ (က) အရ ချမှတ်သည့် အမိန့်ကို ကျေနပ်မှုမရှိပါက စစ်ဆေးရေးအရာရှိချုပ်ထံ ရက်ပေါင်း ၃၀ အတွင်း အယူခံနိုင်သည်။
(ဂ)စစ်ဆေးရေးအရာရှိချုပ်သည် ပုဒ်မခွဲ (ခ) အရ အယူခံမှုကို စိစစ်၍ အလုပ်ရှင်နှင့် အလုပ်သမားတို့ကို ကြားနာပြီး သင့်လျော်သော အမိန့်တစ်ရပ်ရပ်ကို ချမှတ်နိုင်သည်။
(ဃ)စစ်ဆေးရေးအရာရှိချုပ်က ချမှတ်သည့်အမိန့်သည် အပြီးအပြတ်ဖြစ်သည်။

၁၉။(က)စစ်ဆေးရေးအရာရှိသည် ဤဥပဒေပါ ပြဌာန်းချက်တစ်ရပ်ရပ်ကို ကျူးလွန်ဖောက်ဖျက်သူအား စစ်ဆေးရေးအရာရှိချုပ်၏ ခွင့်ပြုမိန့်ဖြင့် သက်ဆိုင်ရာတရားရုံးတွင် ဤဥပဒေအရ တရားစွဲဆိုရမည်။
(ခ)ပုဒ်မခွဲ (က) အရ တရားစွဲဆိုခြင်းကြောင့် အလုပ်သမားက ရရန်ရှိသော အခကြေးငွေများ ရရှိရေးအတွက် တရားမကြောင်းအရ တရားစွဲဆိုတောင်းခံပိုင်ခွင့်ကို မထိခိုက်စေရ။
(ဂ)အလုပ်သမားသည် ပုဒ်မ ၁၂၊ ပုဒ်မခွဲ (ခ) အရ စစ်ဆေးရေးအရာရှိထံသို့ ဖြေရှင်းဆောင်ရွက်ပေးရန် တင်ပြခြင်းမပြုဘဲ နှုတ်ယူဖြတ်တောက်ငွေနှင့် နစ်နာကြေးငွေများကို ပေးချေရန် သက်ဆိုင်ရာ တရားရုံးသို့ တရားမကြောင်းအရ တရားစွဲဆိုနိုင်သည်။
၂၀။တရားရုံးသည် အခကြေးငွေပေးချေရန်ပျက်ကွက်မှု သို့မဟုတ် နှောင့်နှေးကြံ့ကြာမှုမှာ အောက်ဖော်ပြပါ အကြောင်းအရင်းများကြောင့်ဖြစ်သည်ဟု ဆုံးဖြတ်လျှင် အခကြေးငွေပေးချေခြင်းကို ဆိုင်းငံ့ခွင့် သို့မဟုတ် ကင်းလွတ်ခွင့်ပြုနိုင်သည်။
(က)သဘာဝဘေးအန္တရာယ်အပါအဝင် မမျှော်လင့်သောအရေးပေါ်အခြေအနေ ပေါ်ပေါက်လာသဖြင့် အခကြေးငွေကို ပေးချေနိုင်မှုမရှိခြင်း။
(ခ)အလုပ်ရှင်သည် ကြွေးမြီ မပေးဆပ်နိုင်သည့် လူမွဲအဖြစ် ကြေညာခံထားရသူဖြစ်ခြင်း။
(ဂ)သက်ဆိုင်ရာအလုပ်သမားက ကိုယ်တိုင်ဖြစ်စေ၊ ကိုယ်စားလှယ်ဖြင့်ဖြစ်စေ လာရောက်ထုတ်ယူရန် ပျက်ကွက်ခြင်း သို့မဟုတ် ပေးချေသည့်အခကြေးငွေကို လက်ခံရယူရန် ပျက်ကွက်ခြင်း။
၂၁။ပြန်လည်တောင်းခံသော အခကြေးငွေ သို့မဟုတ် ဖြတ်တောက်ထားသည့် အခကြေးငွေများနှင့် စပ်လျဥ်း၍ အောက်ပါကိစ္စတစ်ရပ်ရပ်တွင် တရားစွဲဆိုခြင်းမပြုရ။
(က)စစ်ဆေးရေးအရာရှိထံသို့ လျှောက်ထားသည့်ကိစ္စ မပြီးပြတ်သေးခြင်း။
(ခ)တောင်းခံလျှောက်ထားသူအား ပြန်လည်ထုတ်ပေးရန် ညွှန်ကြားပြိးသည့် ငွေဖြစ်နေခြင်း။
(ဂ)တောင်းခံလျှောက်ထားသူအား ပေးရန်မလိုသောငွေဟု အတည်ပြုပြီးသောငွေ ဖြစ်နေခြင်း။

၁၉၅၁ ခုနှစ်၊ အလုပ်ရုံများ အက်ဥပဒေ

၁၉၅၁ ခုနှစ်၊ အလုပ်ရုံများအက်ဥပဒေကို ပြင်ဆင်သည့်ဥပဒေ
၂၀၁၆ ခုနှစ်၊ ပြည်ထောင်စုလွှတ်တော်အမှတ် - ၁၂

အခန်း (၁)

အသက်ကန့်သတ်ချက်

၂။(က)"ကလေး" ဆိုသည်မှာ အလုပ်လုပ်ကိုင်နိုင်ရန် မှတ်ပုံတင်ဆရာဝန်ကခွင့်ပြုသည့် အသက်တစ်ဆယ့်လေးနှစ်ပြည့်ပြီး အသက်တစ်ဆယ်ခြောက်နှစ် မပြည့်သေးသူကို ဆိုလိုသည်။
(ခ)"လူရွယ်" ဆိုသည်မှာ အသက်တစ်ဆယ့်ခြောက်နှစ်ပြည့်ပြီး အသက်တစ်ဆယ့်ရှစ်နှစ်မပြည့်သေးသူကို ဆိုလိုသည်။
(ဂ)"လူငယ်" ဆိုသည်မှာ "ကလေး" နှင့် "လူရွယ်" ကို ဆိုလိုသည်။
(ဃ)"လူကြီး" ဆိုသည်မှာ အသက်ဆယ့်ရှစ်နှစ် ပြည့်ပြီးသူကို ဆိုလိုသည်။

လုပ်ငန်းစတင်လျှင် အကြောင်းကြားရခြင်း

၈။(၁)လုပ်ငန်းပိုင်ရှင်သည် အလုပ်ရုံအဖြစ် မည်သည့်အဆောက်အဦ၌မဆို စတင်၍ လုပ်ငန်းလုပ်ခြင်း သို့တည်းမဟုတ် အဆောက်အဦကို အသုံးပြုခြင်းမပြုမီ အနည်းဆုံးဆယ့်ငါးရက်ကြိုတင်၍ အလုပ်ရုံနှင့် အလုပ်သမားဥပဒေစစ်ဆေးရေးဦးစီးဌာန၊ ညွှန်ကြားရေးမှူးချုပ်ထံ အောက်ပါအချက်များနှင့် စပ်လျဥ်း၍ နို့တစ်စာပေးပို့ရမည်။
(က)အလုပ်ရုံ၏ အမည်နှင့် တည်နေရာအရပ်။
(ခ)လုပ်ငန်းပိုင်ရှင်၏ အမည်နှင့် နေရပ်အပြည့်အစုံ
(ဂ)အလုပ်ရုံနှင့်စပ်လျဥ်းသည့် အကြောင်းကြားစာများ ပေးပို့ရမည့် နေရပ်အပြည့်အစုံ၊ "ဆက်သွယ်ရန်လိပ်စာ၊ ဖုန်းနံပါတ်၊ ဖက်(စ်)နံပါတ်၊ အီးမေးလ်လိပ်စာ။"
(ဃ)ပြုလုပ်မှုလုပ်ငန်းစဥ်အမျိုးအစား -
(ကက) ဤအက်ဥပဒေအာဏာ တည်သည့်နေ့တွင် ရှိနေပြီးသော အလုပ်ရုံများနှင့် စပ်လျဥ်း၍ ထိုအလုပ်ရုံတွင် လွန်ခဲ့သည့် ဆယ့်နှစ်လအတွင်းလုပ်သည့် ပြုလုပ်မှုလုပ်ငန်းစဥ်။
(ခခ) (အလုပ်ရုံအားလုံးနှင့်စပ်လျဥ်း၍) နောက်လာမည့် ဆယ့်နှစ်လအတွင်းလုပ်မည့် ပြုလုပ်မှု လုပ်ငန်းစဥ်။
(င)လုပ်ငန်းအမျိုးအစား၊ အသုံးပြုရမည့် လျှပ်စစ်ဝန်အားနှင့် စက်စွမ်းအား။
(စ)ဤအက်ဥပဒေအလို့ငှာ အလုပ်ရုံမန်နေဂျာဖြစ်သူ၏ အမည်။
(ဆ)အလုပ်ရုံတွင် ခိုင်းစေဘွယ်ရှိသည့် အလုပ်သမားအရေအတွက်။
(ဇ)ဤအက်ဥပဒေအာဏာတည်သည့်နေ့တွင် ရှိနေပြီးဖြစ်သော အလုပ်ရုံများနှင့် စပ်လျဥ်း၍ လွန်ခဲ့သည့် ဆယ်နှစ်လအတွင်း ပျမ်းမျှခြင်းအားဖြင့် တစ်နေ့လျှင် ခိုင်းစေသည့် အလုပ်သမား အရေအတွက်။
(ဈ)ပြဌာန်းသည့် အခြားအကြောင်းအရာများ။

မန်နေဂျာအသစ်ခန့်ထားလျှင် အကြောင်းကြားရခြင်း

၈။(၄)မန်နေဂျာအသစ်ခန့်ထားသည့်အခါတိုင်း လုပ်ငန်းပိုင်ရှင်သည် မန်နေဂျာအသစ် အလုပ်ဝတ္တရားစတင်ဆောင်ရွက်သည့်နေ့မှ ခုနှစ်ရက်အတွင်း အလုပ်ရုံနှင့် အလုပ်သမားဥပဒေစစ်ဆေးရေးဦးစီးဌာန ညွှန်ကြားရေးမှူးချုပ်ထံ နို့တစ်စာပေးပို့ရမည်။
(၅)မည်သူ့ကိုမျှ အလုပ်ရုံမန်နေဂျာအဖြစ် ခန့်ထားခြင်းမရှိသေးသော ကာလအပိုင်းအခြားအတွင်းသော်လည်းကောင်း၊ မန်နေဂျာအနေဖြင့် ခန့်ထားခြင်းခံရသူသည် အလုပ်ရုံကို အုပ်ချုပ်ခြင်းမပြုသည့် ကာအပိုင်းအခြားအတွင်းသော်လည်းကောင်း၊ မန်နေဂျာအဖြစ် ခေတ္တလုပ်နေသူ သို့တည်းမဟုတ် ယင်းသို့ လုပ်သူမရှိလျှင် လုပ်ငန်းပိုင်ရှင် ကိုယ်တိုင်ပင် ဤအက်ဥပဒေအလို့ငှာ ထိုအလုပ်ရုံ၏ မန်နေဂျာအနေဖြင့် ဖြစ်သည်ဟူ၍ မှတ်ယူရမည်။

အလုပ်ရုံကို ခေတ္တ သို့မဟုတ် အချိန်အကန့်အသတ်မရှိ ပိတ်သိမ်းခြင်းအတွက် အကြောင်းကြားရခြင်း

Signing Error of System Upgrading

Upgrading the system regularly running $ sudo pacman -Syu prevents most signing errors. If delay is unavoidable and system upgrade gets delayed for an extended period, manually sync the pacakge database and upgrade the archlinux-keyring package before system upgrade. Basically it is syncing the package database and installing archlinux-keyring package just before the system upgrade. See the following command;

~
# pacman -Sy archlinux-keyring && pacman -Su

Failed to init transaction (unable to lock database)

Disable Drag Lock DWL

Disabling drag_lock function in dwl is recommended if you are experiencing half a second delay on releasing mouse button simulated by touchpad.

For example, it takes half a second for cursor to reset its shape after doing any drag action like moving or resizing window using touchpad.

To disable that behaviour, it is easy. Just disable static const int drag_lock function by setting its value to 0 in config.h file.

~
/* Trackpad */
static const int tap_to_click = 1;
static const int tap_and_drag = 1;
/* to disable the following drag_lock function, just set its value to 0. Default is 1. */
static const int drag_lock = 0;
static const int natural_scrolling = 0;
static const int disable_while_typing = 1;
static const int left_handed = 0;
static const int middle_button_emulation = 0;
