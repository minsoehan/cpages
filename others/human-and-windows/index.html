လူနှင့် ပြတင်းပေါက်

expand_circle_right December 5, 2025

ငယ်ငယ်က Windows XP ဆိုတာ အတော်ကြားခဲ့ရဖူးတာ မှတ်မိကြလိမ့်မည်။ တော်တော် လူကြိုက်များခဲ့ပုံ ဖြစ်သည်။ အဲ့သည့်နောက် Windows 7၊ ပြီးတော့ Windows 10။ သူတို့ကြားထဲ Windows Vista, Windows 8 ဆိုတာတွေလည်း ရှိခဲ့သည်။ သို့သော် လူကြိုက်များခဲ့ကြပုံမပေါ်။

Windows Version များ တစ်ခုပြီးတစ်ခု ခေတ်ကုန်သွားခဲ့သည်။ ယခု Window 10 လည်း ခေတ်ကုန်သွားခဲ့ပြီ။ Microsoft သည် Windows 10 ကို ၂၀၂၅ ခုနှစ်၊ အောက်တိုဘာလ ၁၄ ရက်နေ့မှစ၍ Support ရပ်လိုက်ပြီဖြစ်ကြောင်း ကြေငြာခဲ့သည်။

Windows 10 has reached the end of support on October 14, 2025. At this point technical assistance, feature updates and security updates is no longer provided. If you have devices running Windows 10, we recommend upgrading them to Windows 11, a more modern, secure, and highly efficient computing experience. If devices do not meet the technical requirements to run on Windows 11, we recommend that you enroll in the Windows 10​​​​​​​ Consumer Extended Security Updates (ESU) program or replace the device with one that supports Windows 11.

ဒါဘာအဓိပ္ပါယ်လဲဆိုလျှင် လက်ရှိ အဆင်ပြေပြေသုံး၍ရနေသော Windows 10 ကွန်ပျူတာများသည် Microsoft ထံမှ လုံခြုံရေးဆိုင်ရာ update များနှင့် အခြား feature အသစ် update များကို ရတော့မည်မဟုတ်ဟူသော သဘောဖြစ်သည်။ ဤသို့ဆိုလျှင် ဘာဖြစ်မည်နည်း? ကွန်ပျူတာ ကျွမ်းကျင်သူဟု အဆိုရှိသူ ဆရာကြီးများ ပြောသည်ကို ကြားဖူး မှတ်သားဖူးကြပေလိမ့်မည်။

သို့သော် လူအများ ဆွေးနွေးငြင်းခုန်ခြင်းမပြုကြသောအချက်မှာ “Microsoft သည် အကောင်းအတိုင်း အလုပ်ဖြစ်နေသော မိမိ၏ Window 10 ကွန်ပျူတာအား သုံးမရတော့အောင် လုပ်ခြင်းဖြင့် မိမိအား နောက်ထပ် ကွန်ပျူတာအသစ်တစ်လုံး ထပ်မံဝယ်ယူဖြစ်စေရန် အကြပ်ကိုင်နေခြင်းလော” ဟူသော အချက်(မေးခွန်း) ပင်ဖြစ်သည်။

ဤသို့ဆိုလျှင် “ခင်ဗျားကို ဘယ်သူက ကွန်ပျူတာအသစ်ဝယ်ခိုင်းနေလို့လဲ Windows 11 ပြောင်းတင်လိုက်ပေါ့” ဟု ပြောကြလိမ့်မယ်။ မှန်ပါသည်။ Windows 11 ပြောင်းတင်လိုက်ရုံသာ။ သို့သော် ဤမျှလောက် ရိုးရှင်းမနေပါ။ Windows 11 ကို ချောချောမွေ့မွေ့ အသုံးပြုနိုင်ဖို့ ကွန်ပျူတာတစ်လုံးတွင် ရှိရမည့် Spec ဟုပြောကြသော Hardware အစိတ်အပိုင်းများ၏ လိုအပ်ချက်မှာ မြင့်မားလှသည်။ Windows 10 ကို အဆင်ပြေပြေ အသုံးပြုနိုင်သော ကွန်ပျူတာများထက် ပို၍ အဆင့်မြင့်ရန် လိုအပ်သည်ကတော့ အမှန်ပင်ဖြစ်သည်။ ထို့အပြင် Windows 11 အသုံးပြုရန် Microsoft Account ရှိရမည်။ ၎င်းလိုအပ်ချက်ကို ယခုအချိန်တွင် ကျော်၍ ရနေသော်လည်း နောင်တွင် မရနိုင်တော့စေရန် Microsoft က ကြိုးပမ်းနေသည်မှာ သေချာသည်။ နီးစပ်ရာ ကွန်ပျူတာကျွမ်းကျင်သူဟု အဆိုရှိသူများအား မေးကြည့်လျှင် သိပါလိမ့်မည်။

သိုးနှင့်တူသောလူ

လူအများစုသည် သိုးများနှင့် တူကြသည်။ ထိုသို့ပြောလျှင် ခံချင်ကြမည်မဟုတ်။ လူသည် မိမိကိုယ်မိမိ ရင့်ကျက်သူ၊ တွေးခေါ်မြှော်မြင်တတ်သူ၊ လိမ္မာပါးနပ်သူ တစ်နေရာရာတွင် တစ်နည်းနည်းဖြင့် သူများထက်သာသူ စသည်ဖြင့် အထင်ရောက်နေကြသူများသာ ဖြစ်သည်။ အခြားသူများက ကိုယ့်ကို ထိုသို့ထင်မှတ်ယုံကြည်စေရန်လည်း ကြိုးပမ်းသည်။ သို့သော် သိုးအုပ်ကို ခြံထဲမောင်းသွင်း၍ ရသကဲ့သို့ လူအများကို ပြတင်းပေါက်ကလေးတစ်ပေါက်ဖောက်ပေးထား၍ ရသည်။ ထိုအပေါက်မှ မြင်ရသမျှကိုသာ အမှန်အဟုတ်ထင်နေစေရန်၊ အကောင်းဆုံးထင်နေစေရန်လည်း လုပ်၍ရသည်။ ထိုလူတို့သည် ထိုပြတင်းပေါက်မှ ကြည့်၍ မြင်ရသမျှကိုသာ အဟုတ်ထင်နေကြလိမ့်မည်။ အခြားအပေါက်များကို ဖွင့်မကြည့်။ တစ်ချို့ကမူ ရှိသည်ဟူ၍ပင် မသိ။

လောကတွင် လူအများက ယုံကြည်လက်ခံကြသည်ဆိုရုံဖြင့် ပကတိအမှန်တရား မဖြစ်နိုင်။ ကိုယ့်အတွက် အကောင်းဆုံး ရွေးချယ်မှု မဖြစ်နိုင်။

Social Media ခေတ်တွင် critical thinking ဘာညာ စသည်ဖြင့် လိုင်းပေါ်တက်၍ ပြောဆိုနေကြသူများကို တွေ့ဖူးကြုံဖူးကြမည်သာ ဖြစ်သည်။ ထိုသူအများစုသည်လည်း Windows 11 တင်ရန် MS Account ဖွင့်ရမည်လား၊ ကွန်ပျူတာအသစ်တစ်လုံးပဲ ပြေးဝယ်ရတော့မည်လား စသည်ဖြင့် အတွေးများစွာဖြင့် ရုန်းကန်နေကြရသူများ ဖြစ်နိုင်သည်။ ၎င်းတို့သည်လည်း ခြံထဲမောင်းသွင်းခံထားရသော သိုးအုပ်ထဲမှ သိုးတစ်ကောင်မျှသာ ဖြစ်ဖို့များသည်။

Windows သည် လူ့သမိုင်းတစ်လျှောက် ဥပမာတစ်ခုမျှသာ ဖြစ်သည်။ နောက်တစ်ခုပြပါဆိုလျင် ဘာသာရေးကို ပြလိုသည်။ လူတစ်ယောက် မည်သည့်ဘာသာဝင်ဖြစ်မည်ဟူသော အချက်သည် ၎င်းမွေးဖွားရာ အရပ်အပေါ်တွင်သာ မူတည်လေ့ရှိသည်။ ဘာသာကြီး ၄ ခုရှိသည့်အနက် တစ်ခုသာအမှန်ဖြစ်၍ ကျန် ၃ ခုမှားသည်ဆိုလျှင် ကမ္ဘာပေါ်ရှိ လူအများစုသည် မိမိမွေးဖွားရာ အရပ်ရှိ ဘာသာတရားကို မှားမှန်းမသိဘဲ ယုံကြည်ကိုးကွယ်နေကြသူများ ဖြစ်နေနိုင်သကဲ့သို့ အခြားတစ်ဘက်ရှိ လူအနည်းစုသည်လည်း မိမိကိုးကွယ်ရာ ဘာသာတရားကို မှန်မှန်းသေချာမသိဘဲ ကိုးကွယ်ယုံကြည်မိနေသူများ ဖြစ်နိုင်သည်။ မည်သည့်ဘာသာတရားက မှား၍ မည်သည့်ဘာသာက မှန်သည်ဟူသော အချက်ကို မည်သူမျှ မသိ။ ဆွေးနွေးငြင်းခုန်ကြခြင်းလည်း မရှိ။ ထိုသို့ဆွေးနွေးငြင်းခုန်ရန် ရှောင်ရှားခြင်းသည်ပင်လျှင် မတူကွဲပြားမှုများကို လက်ခံသူ၊ တစ်ဖက်လူ၏ ယုံကြည်မှုကို လေးစားတတ်သူ၊ အမြင်ကျယ်သူ အစရှိသည်ဖြင့် သရုပ်ဖော်ပြောဆိုကြလေ့ရှိသည်။ ဘာသာတရားများ၏ အမှားအမှန်ကိစ္စ နှိုင်းယှဥ်ချိန်ထိုး ပြောဆိုသူကို မိုက်မဲသူ၊ မလေ့လာသူ၊ ပုန်ကန်သူ၊ ရှေ့နောက်မစဥ်းစားတတ်သူ၊ ဘာမဟုတ်တာကို အကျိုးမရှိငြင်းခုန်နေသူ၊ ငရဲကျမည့်သူ စသည်ဖြင့် လှောင်ပြောင် ရန်လုပ်ကြပေလိမ့်မည်။ မည်သို့ပင်ဆိုစေကာမူ နောက်ဆုံးတွင် ကိုယ်ကိုးကွယ်သည့် ကိုယ့်ဘာသာတရားကသာ အမှန်ဟု တရားသေမှတ်ယူကြသည်သာ ဖြစ်သည်။ ခုခံငြင်းခုန်ကြသည်။ လက်နက်ဆွဲကိုင် အကြမ်းဖက်ကြသည်။ သို့သော် မိမိကိုးကွယ်သည့် ဘာသာတရားကို သေချာလေ့လာ၍ မေးခွန်းထုတ်သူ၊ ခွဲခြမ်းစိတ်ဖြာသုံးသပ်သူ ရှားလှသည်။

ဤသို့ဖြင့် ကျနော်သည်လည်း “အရာရာ အနတ္တ ဟုဆိုလျှင် ဘာက သံသရာလည်နေပါသနည်း” ဟူသော မေးခွန်းအတွက် အဖြေကို နားလည်ဖို့ အခုအထိ ဥာဏ်မမီသေးပါ။
